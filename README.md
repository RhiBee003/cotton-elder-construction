# Cotton Elder Construction

Modern marketing site for **Cotton Elder Construction** — dark mode, responsive layout, and a sleek single-page experience.

## Stack

- Static HTML, CSS, and JavaScript
- No build step required

## Local preview

```bash
cd cotton-elder-construction
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080).

## Deploy

Host on GitHub Pages, Netlify, Render, or any static host. Point the site root at this folder.

## Customize

- Update contact details in `index.html`
- Swap project copy and placeholder photography
- Wire the contact form to Formspree, Netlify Forms, or your backend
